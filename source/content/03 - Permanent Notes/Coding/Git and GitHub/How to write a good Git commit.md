## Summary

- Group related commits together
- Use imperative verb forms
	- “Fix” instead of “Fixed”
- Think: “If applied, my commit will (INSERT COMMIT MESSAGE TEXT)
- Conventional commits

| Commit type | Description                          |
| ----------- | ------------------------------------ |
| Feat        | Feature                              |
| Fix         | Bug fixes                            |
| Docs        | Changes to the documentation; README |
| Style       | Style or formatting change           |
| Perf        | Improves code performance            |
| Test        | Test a feature                       |
| Closes      | Closes a ticker (w/number)           |
> CLI Command for detailed commit
> `git commit -m <Summary> -m <Description>`

> CLI Command to edit the most recent commit message
> `git commit --amend -m “new commit message”`
